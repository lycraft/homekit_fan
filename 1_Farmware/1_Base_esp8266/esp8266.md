## ESP8266

#### esp8266引脚对应

| ESP_GPIO | ESP_Char | Use            | BOOT |
| -------- | -------- | -------------- | ---- |
| GPIO16   | D0       | LED            | 高   |
| GPIO5    | D1       | 继电器(总开关) |      |
| GPIO4    | D2       | 继电器(低速)   |      |
| GPIO0    | D3       | 按键总开关     | N低  |
| GPIO2    | D4       | 按键低速       | 高   |
| GPIO14   | D5       | 继电器(中速)   |      |
| GPIO12   | D6       | 继电器(高速)   |      |
| GPIO13   | D7       | 按键中速       |      |
| GPIO15   | D8       | 按键高速       | N高  |
| ADC      | A0       |                |      |
